In mathematics, a topology on a set \(X\) is a collection \(\mathcal{T}\) of subsets of \(X\) that satisfies the following three properties:

  1. \(X\) and the empty set \(\emptyset\) are in \(\mathcal{T}\).
  2. \(\mathcal{T}\) is closed under arbitrary unions, which means if \(\{U_\alpha\}\) is a collection of sets in \(\mathcal{T}\), then \(\bigcup_{\alpha} U_\alpha \in \mathcal{T}\).
  3. \(\mathcal{T}\) is closed under finite intersections, which means if \(U_1, U_2, \ldots, U_n\) are sets in \(\mathcal{T}\), then \(U_1 \cap U_2 \cap \cdots \cap U_n \in \mathcal{T}\).

A set \(X\) equipped with a topology \(\mathcal{T}\) is called a topological space. The sets in \(\mathcal{T}\) are called open sets, and the topology \(\mathcal{T}\) essentially describes a structure on \(X\) that allows for the definition of concepts such as continuity, convergence, and boundary.
